# Start Here

1. Copy this package into the root of your Software Lab IBRA project repository.
2. Open the repository in Codex.
3. Run the prompt in `codex_prompts/initial_setup_prompt.md`.
4. Add new project updates in natural language to `project_progress/natural_language_inbox.md`.
5. Create automations using the prompts in `codex_prompts/automation_prompts.md`.

Recommended automation schedule:

| Automation | Suggested Time |
|---|---|
| Tuesday pre-meeting report | Tuesday 12:00 Europe/Berlin |
| Tuesday post-meeting planning | Tuesday 18:30 Europe/Berlin |
| Saturday progress push | Saturday 10:00 Europe/Berlin |

Minimum useful automation setup:

1. Tuesday pre-meeting report
2. Saturday progress push

You can add Tuesday post-meeting planning later after the group is used to writing meeting notes into `natural_language_inbox.md`.
